# getGreWordListTapes

用`nodejs`批量下载GRE词汇精选（红宝书）音频到本地，方便离线复习

=====================

# 使用

## clone

``` 
git clone https://github.com/dukewan/getGreWordListTapes.git
```

## 安装依赖

```
npm install // 安装失败的话，请 sudo npm install
```

## 运行

```
// 进入项目目录
cd getGreWordListTapes

// 运行
node index.js
```
